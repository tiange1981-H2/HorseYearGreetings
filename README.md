# 马年定制祝福 H5

这是一个轻量级的马年祝福生成页面。用户输入姓名后，可以生成包含“Power to X”（绿电/氢能）科技主题的专属拜年图片。

## 功能特点
- **随机祝福**：随机选择马年吉祥话。
- **随机背景**：随机展示风电、光伏或绿色化工厂背景。
- **本地生成**：基于 HTML5 Canvas，无需后端服务器，保护隐私。

## 部署
本项目设计为静态网页，可直接托管于 GitHub Pages、Vercel 或 Netlify。

## 本地开发
直接在浏览器中打开 `index.html` 即可预览（需注意本地文件的 Canvas 安全限制，已做降级处理）。
推荐使用 Live Server 启动。
